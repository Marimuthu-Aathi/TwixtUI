# TwixtUI
'Tw' from Tailwind and 'ixt' suggesting a mix and integration

# React App

```
npm install -g postcss-cli
```

# Component Supported.

1. Hamburger Menu
2. UserMenu
3. SimpleTable
4. Heading (H1 to H6)
5. Text for Paragraph
6. Switch Input
7. Text area
8. <icon (Edit Delete icons)
9. Dialog
10. Modal
11. Simple Button and Submit Button
12. SingleLineInput
13. ToolName
